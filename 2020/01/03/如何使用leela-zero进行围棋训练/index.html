leela zero简介

leela zero是由比利时程序员Gian-Carlo_Pascutto 对 AlphaGo zero 的论文 Mastering the game of Go without human knowledge（无须人类知识掌握围棋）中内容的开源实现。

人机对弈环境搭建

根据leela zero github的README中的指示
如果想用自己的硬件帮助训练增强Leela zero，可前往此处下载一个自动训练的程序AutoGTP，它将自动地连接服务器并在后台进行它的工作，在每次训练后上传结果，若想停止训练只需要关闭窗口即可。另外再说一句，我们还可以通过诸如谷歌和微软等大公司提供的云计算服务来帮助leela zero这个项目。
如果只是想和它进行对弈，那么你需要三样东西：一个是引擎，相当于leela zero的身体，在此处下载；另一个是权重文件，相当于leela zero的大脑，在官网上的最下面Best Network Hash的表中我们可以选取最新的权重文件下载，应该说最新的也代表着目前的最强版本（从旁边的Test Matches的表中，我们似乎可以看出它是通过两个权重文件进行自对弈，当新的权重文件胜率超过55%就替换老版本）；最后一个是图形化界面，相当于给leela zero穿的衣服，这里我选择了具有好看的界面的Sabaki。

为Sabaki设置leela zero的引擎
点击Sabaki界面的Engines菜单栏中的Manage Engines，下方出现一个界面后点击add添加引擎。点击小文件夹图标，找到之前下载的leelaz.exe引擎所在的路径，后面需要设置相关参数（根据个人的硬件配置来设置），具体可参考leela zero参数及解释。这里我的参数设置如下：-g -t4 -w 40b_257a_64k_q –noponder –playouts 3000(这里-w后的文件名即为之前下载的权重文件，需要将其放到leela zero的文件夹下)；initial commands命令如下：time_settings 0 20 1(意为每步棋leela zero思考20s)
